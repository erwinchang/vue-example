## vite example 


進入vue3之後，建議改用vite的方式或vue-clicd v
Vite 通过在一开始将应用中的模块区分为 依赖 和 源码 两类，改进了开发服务器启动时间
```
npm init vite-app vite-ex01
```

執行後台
```
npm run dev
```

重新build
```
npm run build
```
